All art is original and made in microsoft paint.

All music is original.

UI objects use default sprites.

UI functions using three scripts to control all buttons, dropdowns, inputfields, text fields.

DD_Controller:
-Controls all DropDowns
-Controls JSON output
-Contains dice rolling algorithm
-Outputs dice values
-Sends dice values to attribute dropdowns

UI_Controller
-Controls Play, Generate Character, Back, and Quit button functionality.

Input_Controller
-Controls the changed values of input fields
